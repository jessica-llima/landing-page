# Frontend (Vite + React) - Landing Page

Scaffold mínimo com Vite + React.

Para rodar (pré-requisitos: Node.js >= 18):

```powershell
cd frontend
npm install
npm run dev
```

O dev server abrirá tipicamente em `http://localhost:5173`.

O formulário demonstra envio para `http://localhost:8001/api/submit` (backend FastAPI scaffold).
